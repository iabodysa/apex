# Apex Habitat 0.8.0

This release adds supplier cost recovery, a system-monitoring workspace, native
operational notifications, hand-over print documents, an occupancy-history
engine, and dashboard insights — plus broad data-integrity, security, and
performance hardening. Run `bench migrate` after pulling.

## What changed for users

- **Supplier Cost Recovery** — tag an Accommodation Assignment as an external
  supplier worker; the daily ledger tracks the cost and the new "Supplier Cost
  Recovery" report sums each month per supplier and applies a configurable
  operational markup (set in Habitat Settings).
- **Operational notifications go native** — license/lease expiry and overdue
  tasks now post a timeline comment on the source document and can email through
  Frappe Notifications, controlled by a new "Enable Operational Notifications"
  setting. The old custom alert list is retired for new alerts.
- **Hand-over documents** — five bilingual (Arabic/English) print formats:
  custody issue/return receipts, damage/deduction notice, checkout clearance,
  and the maintenance Work Order job sheet (with signature blocks).
- **Occupancy history** — buildings now keep a daily occupancy snapshot, so the
  new "Occupancy Trend" report can show min/average/max occupancy and days over
  capacity over any period (previously only today's figure existed).
- **Dashboard insights** — new number cards and charts on the workspaces (cost
  this month, supplier-billed cost, assets moved, tasks due/completed, occupancy
  and ledger-cost trends).
- **Cleaner forms** — list views, quick filters, and readable titles were added
  across the main forms; the Room generator now updates existing room types on
  re-run and confirms before adding new rooms.
- **Maintenance & custody integrity** — duplicate utility bills, overlapping
  leases, duplicate maintenance work orders, and over-quantity custody returns
  are now rejected.

## What changed under the hood

- A new System Administration workspace surfaces the background engines
  (Accommodation Ledger, Occupancy Snapshot, asset movements) and framework logs
  for administrators; the workspaces were consolidated to reduce clutter.
- Read-only background-engine doctypes for system-written records; per-document
  permission checks tighten security on several actions.
- Report queries were optimised for performance (N+1 lookups removed), and the
  full Arabic interface translation was completed.

## If you are updating an existing site

Run `bench migrate` and clear cache after pulling this release. On the next Desk
