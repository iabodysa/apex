# Apex Habitat 0.8.0

This release lets you bill suppliers back, watch occupancy trends, print every handover,
and keep an eye on the whole operation from one place — plus security and reliability
improvements. Run `bench migrate` after pulling.

## What changed for users

- **Recover what suppliers' workers cost you** — tag an Accommodation Assignment as an
  external supplier worker, and the new "Supplier Cost Recovery" report totals each
  supplier's monthly cost and adds your markup (set in Habitat Settings), ready to bill.
- **Alerts where you work** — licence and lease expiry and overdue tasks now post right
  on the document's own timeline and, if you like, land in your inbox, all controlled by
  a single "Enable Operational Notifications" setting.
- **Print a signed handover in seconds** — five bilingual (Arabic/English) documents:
  custody issue and return receipts, a damage/deduction notice, checkout clearance, and
  the maintenance work-order job sheet, each with signature blocks.
- **See how full your buildings have been** — buildings now keep a daily occupancy
  snapshot, so the new "Occupancy Trend" report charts the lows, averages, highs and days
  over capacity across any period.
- **Insights at a glance** — new cards and charts across the workspaces for cost this
  month, supplier-billed cost, assets moved, tasks due and completed, and occupancy and
  cost trends.
- **Cleaner forms** — friendlier list views, quick filters and readable titles across the
  main forms; the room generator now updates existing room types on re-run and confirms
  before adding new rooms.
- **Confident data entry** — Apex now guards against duplicate utility bills, overlapping
  leases, duplicate work orders and over-quantity custody returns, so your records stay
  clean.

## What changed under the hood

- A new System Administration workspace gives administrators a single window onto the
  background engines (accommodation ledger, occupancy snapshots, asset movements) and
  framework logs, and the workspaces were tidied to reduce clutter.
- Security & reliability improvements, with report queries tuned for speed and the full
  Arabic interface translation completed.

## If you are updating an existing site

Run `bench migrate` and clear cache after pulling this release. On the next Desk boot,
Frappe will read this change log and show the update dialog once per user.
