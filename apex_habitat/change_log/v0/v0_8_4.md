# Apex Habitat 0.8.4

This release lets the right reports, reminders and follow-ups send themselves. No change
to the room generator or other flows.

## What changed for users

- **Reports that send themselves** — Supplier Cost Recovery (monthly), Occupancy Trend,
  Maintenance Backlog and Scheduled Task Compliance (weekly) can now land in inboxes on
  schedule. Shipped ready to switch on: set your real recipients and enable.
- **Polished emails in a click** — ready-made templates for building licence renewal,
  lease renewal and resident-request acknowledgement, right from the Email action on
  those documents.
- **Nothing slips through the cracks** — assigning a resident request drops a follow-up
  to-do into that person's queue, and resolving or closing the request ticks it off
  automatically.
- **Your key workspaces, one click away** — the Help menu now jumps straight to the
  Command Center and Setup.

## What changed under the hood

- Report emails arrive switched off, so an update never starts emailing until you've set
  your recipients. The new navbar links add to your existing Help menu without disturbing
  it.
- Follow-up to-dos reuse Frappe's native assignment and add no overhead. The workspace
  layout is unchanged. Security & reliability improvements.

## If you are updating an existing site

Run `bench migrate` and clear cache after pulling. The Auto Email Reports, Email Templates
and navbar links are seeded automatically (reports disabled). On the next Desk boot,
Frappe shows this update dialog once per user.
