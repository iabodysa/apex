# Apex Habitat 0.7.2

This patch makes reports faster, completes the Arabic interface, and ships security and
reliability improvements. No data migration is required.

## What changed for users

- Your Checkout Pending Clearance and cleaning reports open noticeably faster.
- Trust your numbers — occupancy percentage and asset depreciation update live as your
  data changes.
- Work in Arabic end to end — the full interface is now translated.
- Security & reliability improvements.

## What changed under the hood

- Security & reliability improvements, plus performance tuning on the daily cost
  allocation.

## If you are updating an existing site

Run `bench migrate` and clear cache after pulling this release. On the next Desk boot,
Frappe will read this change log and show the update dialog once per user.
