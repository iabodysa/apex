# Apex 1.4.1

A maintenance release with security and reliability improvements.

## What changed for users

- Tighter access — people see and act on only the records they are entitled to.
- More dependable approvals and fuel-console actions.
- Stronger input validation across forms and requests.

## What changed under the hood

- Security & reliability improvements.

## If you are updating an existing site

Pull, then `bench migrate` and clear cache. No data migration is required.
