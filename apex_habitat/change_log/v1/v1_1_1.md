# Apex 1.1.1

A maintenance release that makes the test and continuous-integration suite pass
cleanly and hardens one endpoint.

## What changed for users

- No functional changes. This release is internal quality and reliability work.

## What changed under the hood

- The Rental Settlement "create payment request" endpoint is now restricted to
  POST, closing a cross-site-request risk on a state-changing call.
- Removed an unused import and annotated a system-managed ledger cleanup so the
  lint and permission-guard checks pass.
- Made the changelog-popup test version-agnostic so it no longer asserts wording
  from an older release.

## If you are updating an existing site

Pull and clear cache. No migration is required.
