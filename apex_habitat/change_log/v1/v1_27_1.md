# Apex 1.27.1

Small navigation fixes so nothing built is left hidden.

## What changed for users

- **Safety onboarding is back.** Merging Maintenance and Safety into the new
  Facilities workspace had dropped the Safety getting-started steps; they are
  restored on Facilities.
- **Cost by Dimension is reachable.** The report is now linked in the Costs
  workspace (it was built but not surfaced anywhere).
- **Arrivals Desk is reachable.** A shortcut to the new Arrivals Desk now sits in
  the Accommodation workspace.

## What changed under the hood

- Re-added the onboarding content block to the Facilities workspace and added the
  missing report/page links to the Costs and Accommodation workspaces; `modified`
  bumped so migrate re-imports them. "Wire every new DocType / Page / Report into
  its workspace" is now a standing rule.

## If you are updating an existing site

- Run `bench migrate`, then `bench clear-cache`.
